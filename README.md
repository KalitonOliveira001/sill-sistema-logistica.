
# SILL — Sistema Inteligente de Logística Local

Sistema de logística para Quixadá-CE com integração ao Google Sheets.

## Tecnologias
- React + TypeScript + Vite
- Tailwind CSS
- Leaflet (mapa de entregas)
- Google Sheets API (banco de dados)

## Funcionalidades
- Login por PIN para Gestores e Entregadores
- Painel do gestor com mapa de entregas em tempo real
- Tela do entregador com status das entregas
- Rastreamento de pedidos para clientes
- Ciclo mensal automático de renovação de dados
